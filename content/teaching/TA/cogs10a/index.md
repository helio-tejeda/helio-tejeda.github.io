## Course Description

Exploratory data analysis. The main theme of this course is that you use computer programs to
manipulate data so they tell a coherent story. You will learn all sorts of ways of plotting data (this
is called data visualization), how to transform data, how to look for consistencies and patterns in the
data, and how to interpret these patterns to reach conclusions.

Introduction to R and computer programming. R is a high-level language. It is more involved than
applications like SPSS and Excel. But it is easier than C or Python, and is by far the best language
for visualizing data.
